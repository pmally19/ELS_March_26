HOW TO USE (USER GUIDE):
======================================================================================

STEP 1: VIEW COST CENTERS
--------------------------
1. Navigate to http://localhost:5173/controlling
2. Click "Cost Centers" tab
3. See all 12 sample cost centers with aggregated data

STEP 2: CREATE PLANNING
------------------------
1. Click "Planning" tab
2. Select cost center from dropdown
3. Enter GL account (e.g., 5000001)
4. Enter planned amount
5. Optionally select activity type
6. Click "Create Planning"
7. Success toast appears

STEP 3: POST ACTUAL COSTS
--------------------------
1. Click "Post Actuals" tab (NEW!)
2. Select cost center from dropdown
3. Select posting date
4. Enter GL account
5. Enter actual amount
6. Optionally select activity type
7. Click "Post Actual Cost"
8. Success toast appears

STEP 4: VIEW VARIANCE
----------------------
1. Click "Variance Analysis" tab
2. Select year and period from dropdowns
3. View plan vs actual variance
4. See color-coded variance percentages
   - Green: ≤5% variance (On Target)
   - Yellow: 5-10% variance (Watch)
   - Red: >10% variance (Review)

STEP 5: VIEW PROFITABILITY
---------------------------
1. Click "Profitability (CO-PA)" tab
2. Select year and period
3. View real profitability data (if profit center actuals exist)
4. See revenue, COGS, margins by profit center

STEP 6: MONITOR OVERVIEW
-------------------------
1. Click "Overview" tab
2. See summary cards:
   - Active Cost Centers
   - Total Planned Costs
   - Total Actual Costs
   - Variances >10%
3. View top 5 cost centers by variance
4. View top 5 profit centers by revenue
