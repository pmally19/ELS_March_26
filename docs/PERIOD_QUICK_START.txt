## 🎓 HOW IT WORKS

### Period Lock Check Flow:
```
User posts document
    ↓
Middleware intercepts
    ↓
Extract: company_code_id, posting_date
    ↓
Calculate: period = month, fiscalYear = year
    ↓
Query: posting_period_controls
    ↓
Check: status = LOCKED? → REJECT ❌
Check: status = CLOSED && !allow_posting? → REJECT ❌
    ↓
Query: fiscal_periods
    ↓
Check: posting_allowed = FALSE? → REJECT ❌
    ↓
All checks pass → ALLOW ✅
```

### Period Closing Flow:
```
User clicks "Close Period"
    ↓
Create period_end_closing record
    ↓
Call /process endpoint
    ↓
Validate GL entries balanced
    ↓
Count entries, calculate totals
    ↓
All balanced? 
  YES → Update fiscal_periods.status = 'Closed'
      → Set posting_allowed = FALSE
      → Status = 'completed' ✅
  NO  → Status = 'failed' ❌
    ↓
Return validation results
```
