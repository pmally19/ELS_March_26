# JuneERP - Complete Enterprise Resource Planning System

## Overview
JuneERP is a comprehensive Enterprise Resource Planning (ERP) system built with modern web technologies.

## Features
- Finance & Accounting
- Sales & Distribution
- Materials Management
- Human Resources
- Production Planning

## Tech Stack
Frontend: React + TypeScript + Tailwind  
Backend: Node.js + Express  
Database: PostgreSQL

## Run the Project
npm install
npm run dev
